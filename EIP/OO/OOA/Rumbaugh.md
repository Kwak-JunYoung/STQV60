- 소프트웨어 구성 요소를 그래픽 표기법을 이용하여 모델링하는 [[OOA]] 기법

1. Object Modeling
	- 객체 모델링
	- [[Object Diagram]]
	- 정보 모델링이라고도 하며 시스템에서 요구하는 객체를 찾고 객체들 간의 관계를 정의
2. Dynamic Modeling
	- 동적 모델링
	- [[State Diagram]]
	- 시간의 흐름에 따라 객체들 사이의 제어 흐름, 동작 순서 등의 동적인 행위를 표현
3. Functional Modeling
	- 기능 모델링
	- [[DFD]]
	- 프로세스들의 자료 흐름을 중심으로 처리 과정 표현
