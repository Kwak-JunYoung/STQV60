일반적으로 `CUME_DIST()` 함수는 정렬된 컬럼을 기반으로 작동합니다. 이 함수는 현재 행보다 작거나 같은 값의 수를 현재 행보다 크거나 같은 전체 값 수로 나눈 값을 반환합니다. 즉, 누적 분포 함수입니다.

| Value | CUME_DIST |       |
| ----- | --------- | ----- |
| 10    | 0.2       | (1/5) |
| 15    | 0.4       | (2/5) |
| 20    | 0.8       | (4/5) |
| 20    | 0.8       | (4/5) |
| 30    | 1.0       | (5/5) |
